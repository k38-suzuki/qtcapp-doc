
設定の読み込みと保存
====================

ここでは、設定したパラメータの読み込みと保存の仕方を説明します。

設定の読み込み
--------------

以下の手順で設定を読み込みます。

1. メニューの「File」を開く。
2. 「Import」を選択する。
3. 表示されるダイアログでファイルを選択する。
4. ダイアログの「Open」ボタンを押す。

設定の保存
----------

以下の手順で設定を保存します。

1. メニューの「File」を開く。
2. 「Export」を選択する。
3. 表示されるダイアログにファイル名を入力する。
4. ダイアログの「Save」ボタンを押す。
